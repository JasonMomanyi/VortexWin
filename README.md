## Launching Instructions

To launch the project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/JasonMomanyi/VortexWin.git
   cd VortexWin
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## Build Information

To build the project for production, run:
```bash
npm run build
```
This creates an optimized build of the application in the `build/` directory.
